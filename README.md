# ruth-and-nora-dressup
Dress Up app for spending points on outfits/ accessories for the babies to eventually integrate with habit-tracker

# Sprites
Sprite sheets are placed in `/src/sprites`. To start there are 2 sprite sheets, one for Nora and one for Ruth with a set of idle sprites and a set of animated sprites

# Guidance
### Economy
The general economic guidance is that a user can earn about 1000-2000 points per day, so most items should be within that range except very rare cool items that may be more expensive

Changing the color of a garment or object should be fairly cheap

### Management
We can set up a DB with an inventory table to store all of the things a user owns and then filter this by logged in user 

### Items
- Items can range from hats (easiest) to collars to clothes like shirts and dresses (CLOTHING)
- Items can also range from wallpaper (background png) to room items which can be animal objects like water/food bowls and scratching posts / balls as well as more human items like chairs and beds and paintings on the walls
